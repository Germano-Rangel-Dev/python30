Dia 01 – O que é Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Python é uma linguagem de programação de alto nível criada com o objetivo de facilitar o desenvolvimento de software. Diferente de linguagens mais antigas e complexas, Python foi projetado para ser simples de ler, escrever e manter. Isso faz com que o programador possa focar na lógica do problema, e não em detalhes técnicos excessivos da linguagem.

Uma linguagem de programação serve como meio de comunicação entre o ser humano e o computador. Python atua como um intermediário: você escreve instruções em uma sintaxe próxima da linguagem humana, e o interpretador Python traduz essas instruções para o computador executar.

Conceito-chave:
Python é uma linguagem interpretada, de alto nível e com tipagem dinâmica, utilizada para resolver problemas reais de forma simples e eficiente.

2️⃣ Explicação Técnica Detalhada

Python é classificado como uma linguagem interpretada, o que significa que o código é executado linha por linha por um interpretador. Não existe a etapa obrigatória de compilação antes da execução, como ocorre em linguagens como C ou Java.

Outra característica importante é a tipagem dinâmica. Em Python, você não precisa declarar o tipo de uma variável. O próprio interpretador identifica o tipo com base no valor atribuído. Isso torna o código mais curto, mas exige atenção para evitar erros lógicos.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Básico


print("Olá, mundo!")

Este é o programa mais simples possível em Python. A função print serve para exibir informações na tela. Tudo que está entre aspas é tratado como texto.

Exemplo 2 – Intermediário


nome = "Germano"
print("Olá,", nome)

Aqui introduzimos uma variável. A variável nome armazena um texto e depois é usada junto com o print.

Exemplo 3 – Avançado


nome = "Germano"
idade = 30
print(f"Olá, meu nome é {nome} e tenho {idade} anos.")

Neste exemplo usamos uma f-string, que permite misturar texto com variáveis de forma organizada e legível.
