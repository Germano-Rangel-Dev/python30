Dia 09 – Laços de Repetição em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Em muitos programas, é necessário executar o mesmo bloco de código várias vezes. Repetir manualmente o mesmo código não é eficiente e torna o programa difícil de manter. Para resolver isso, utilizamos os laços de repetição.

Laços permitem que um conjunto de instruções seja executado repetidamente, enquanto uma condição for verdadeira ou enquanto houver elementos a percorrer. Em Python, os principais laços são while e for.

Conceito-chave:
Laços de repetição permitem executar um bloco de código várias vezes de forma controlada e automática.

2️⃣ Explicação Técnica Detalhada

O laço while executa um bloco de código enquanto uma condição for verdadeira. É importante garantir que essa condição em algum momento se torne falsa, caso contrário o programa entrará em um loop infinito.

Já o laço for é usado para percorrer sequências, como listas, intervalos de números ou textos. Ele é mais seguro e previsível para a maioria dos casos.

A função range() é frequentemente utilizada com o for para gerar sequências numéricas.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Laço while


contador = 1

while contador <= 5:
    print(contador)
    contador += 1

O laço será executado enquanto contador for menor ou igual a 5. A cada repetição, o valor é incrementado.

Exemplo 2 – Laço for com range


for i in range(1, 6):
    print(i)

O range(1, 6) gera números de 1 a 5. O laço percorre automaticamente essa sequência.

Exemplo 3 – Percorrendo texto


nome = "Python"

for letra in nome:
    print(letra)

Neste exemplo, o laço percorre cada caractere da string.
