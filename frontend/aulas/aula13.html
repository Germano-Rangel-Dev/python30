Dia 13 – Funções em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

À medida que os programas crescem, repetir código se torna um grande problema. Códigos repetidos são difíceis de manter, propensos a erros e tornam o projeto confuso. Para resolver isso, utilizamos funções.

Funções permitem agrupar um conjunto de instruções sob um nome, possibilitando reutilização, organização e clareza. Elas são um dos pilares da programação estruturada e estão presentes em praticamente todo software profissional.

Conceito-chave:
Função é um bloco de código reutilizável que executa uma tarefa específica quando chamado.

2️⃣ Explicação Técnica Detalhada

Em Python, funções são definidas com a palavra-chave def, seguida do nome da função e parênteses. Dentro dos parênteses, podem ser definidos parâmetros.

Parâmetros permitem que a função receba dados externos. Quando a função é executada, esses dados são chamados de argumentos.

Uma função pode ou não retornar um valor usando a palavra-chave return. Quando não há retorno, a função apenas executa ações.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Função simples


def saudacao():
    print("Olá, seja bem-vindo!")

saudacao()

A função saudacao executa sempre o mesmo bloco de código.

Exemplo 2 – Função com parâmetros


def saudacao(nome):
    print("Olá,", nome)

saudacao("Germano")

Agora a função recebe um valor externo através do parâmetro nome.

Exemplo 3 – Função com retorno


def soma(a, b):
    return a + b

resultado = soma(10, 5)
print(resultado)

O valor calculado dentro da função é retornado e armazenado em uma variável.

7️⃣ Resolução dos Desafios


# Desafio 1

    print("Germano")
