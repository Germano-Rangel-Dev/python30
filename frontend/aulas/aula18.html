Dia 18 – Programação Orientada a Objetos

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

À medida que aplicações crescem, organizar o código apenas com funções se torna insuficiente. Surge a necessidade de modelar o sistema de acordo com o mundo real, agrupando dados e comportamentos relacionados.

A Programação Orientada a Objetos (POO) resolve esse problema ao permitir criar objetos que representam entidades reais, como usuários, produtos ou contas bancárias, contendo tanto dados quanto comportamentos.

Conceito-chave:
POO é um paradigma de programação que organiza o código em objetos que possuem atributos (dados) e métodos (ações).

2️⃣ Explicação Técnica Detalhada

Em Python, objetos são criados a partir de classes. Uma classe é um molde que define como os objetos serão criados.

Dentro de uma classe, utilizamos:

  • atributos – variáveis que pertencem ao objeto
  • métodos – funções que pertencem ao objeto

O método especial __init__ é chamado automaticamente quando um objeto é criado e serve para inicializar os atributos.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Criando uma classe simples


class Pessoa:
    def __init__(self, nome):
        self.nome = nome

p = Pessoa("Germano")
print(p.nome)

Criamos uma classe e instanciamos um objeto a partir dela.

Exemplo 2 – Classe com múltiplos atributos


class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

p = Pessoa("Germano", 30)
print(p.nome, p.idade)

O objeto agora possui mais informações associadas.

Exemplo 3 – Classe com métodos


class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def apresentar(self):
        print("Olá, meu nome é", self.nome)

p = Pessoa("Germano", 30)
p.apresentar()

Métodos permitem definir comportamentos associados ao objeto.

        self.modelo = modelo

        print("Modelo:", self.modelo)

# Desafio 3
class ContaBancaria:
    def __init__(self, saldo):
        self.saldo = saldo

    def depositar(self, valor):
        self.saldo += valor

    def sacar(self, valor):
        if valor <= self.saldo:
            self.saldo -= valor
