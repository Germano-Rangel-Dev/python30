Dia 21 – Projeto Prático: Login e Autenticação

Curso Python em 30 Dias

1️⃣ Visão Geral do Projeto

Depois de criar um sistema de cadastro, o próximo passo natural é controlar o acesso ao sistema. Em aplicações reais, usuários precisam se identificar para acessar recursos protegidos. Esse processo é chamado de autenticação.

Neste projeto, você irá implementar um sistema simples de login, utilizando verificação de credenciais e boas práticas básicas de segurança, preparando o terreno para sistemas web e APIs.

Objetivo do projeto:
Criar um sistema de login que valide email e senha antes de permitir acesso.

2️⃣ Conceitos Importantes

  • Autenticação: confirmar quem é o usuário
  • Credenciais: email e senha
  • Hash de senha: forma segura de armazenar senhas

Nunca se deve armazenar senhas em texto puro. Em vez disso, usamos funções de hash para proteger os dados.

3️⃣ Implementação Passo a Passo

Passo 1 – Gerando hash de senha


import hashlib

def gerar_hash(senha):
    return hashlib.sha256(senha.encode()).hexdigest()

A função transforma a senha em um valor irreversível.

Passo 2 – Cadastro com senha


class Usuario:
    def __init__(self, nome, email, senha):
        self.nome = nome
        self.email = email
        self.senha = gerar_hash(senha)

    def to_dict(self):
        return {
            "nome": self.nome,
            "email": self.email,
            "senha": self.senha
        }
