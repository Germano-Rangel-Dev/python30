Dia 02 – Instalação do Python e Preparação do Ambiente

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Antes de escrever programas mais complexos, é essencial preparar corretamente o ambiente de desenvolvimento. Um ambiente mal configurado é uma das maiores fontes de frustração para iniciantes, pois gera erros que não estão ligados à lógica do código, mas sim à instalação incorreta de ferramentas.

Programar envolve não apenas escrever código, mas também garantir que o interpretador, o editor e o sistema operacional estejam trabalhando em harmonia. Neste dia, você aprenderá exatamente como preparar esse ambiente de forma correta e profissional.

Conceito-chave:
Ambiente de desenvolvimento é o conjunto de ferramentas (Python, editor, terminal e bibliotecas) necessárias para criar e executar programas.

2️⃣ Explicação Técnica Detalhada

Quando você instala o Python, está instalando o interpretador responsável por executar arquivos com extensão .py. Esse interpretador lê o código linha por linha e transforma suas instruções em ações que o computador consegue executar.

Além do interpretador, a instalação inclui o pip, que é o gerenciador de pacotes do Python. O pip permite instalar bibliotecas externas, ampliando as capacidades da linguagem.

Outro ponto crítico é o PATH do sistema operacional. O PATH define onde o sistema irá procurar programas executáveis. Se o Python não estiver no PATH, o comando python não será reconhecido no terminal.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Verificando a instalação


python --version

Este comando exibe a versão do Python instalada. Se o comando funcionar, significa que o Python está corretamente configurado no PATH.

Exemplo 2 – Executando um arquivo Python


python teste.py

Aqui executamos um arquivo Python diretamente pelo terminal. Esse é o método mais comum de rodar scripts e aplicações.

Exemplo 3 – Usando o terminal integrado do VS Code


print("Ambiente configurado com sucesso!")

No VS Code, o terminal integrado permite escrever, executar e depurar código no mesmo ambiente, aumentando produtividade e organização.
