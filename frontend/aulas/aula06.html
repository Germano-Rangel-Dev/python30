Dia 06 – Entrada de Dados em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Até agora, todos os programas que criamos tinham valores definidos diretamente no código. Embora isso seja útil para aprender, programas reais precisam interagir com o usuário, recebendo informações durante a execução.

A entrada de dados permite que um programa se torne dinâmico, adaptando seu comportamento conforme as informações fornecidas pelo usuário. Em Python, essa funcionalidade é realizada principalmente pela função input().

Conceito-chave:
Entrada de dados é o processo de receber informações externas (geralmente do usuário) durante a execução de um programa.

2️⃣ Explicação Técnica Detalhada

A função input() pausa a execução do programa e aguarda que o usuário digite algum valor no teclado. Tudo o que o usuário digitar será retornado pela função como um texto (str).

Esse detalhe é extremamente importante: mesmo que o usuário digite um número, o Python interpretará esse valor inicialmente como texto. Para realizar operações matemáticas, é necessário converter o valor para o tipo correto utilizando funções como int() ou float().

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Entrada simples de texto


nome = input("Digite seu nome: ")
print("Olá,", nome)

Neste exemplo, o programa solicita o nome do usuário e depois exibe uma mensagem personalizada.

Exemplo 2 – Entrada de números


idade = input("Digite sua idade: ")
idade = int(idade)
print("Você tem", idade, "anos")

Aqui ocorre a conversão do texto digitado para um número inteiro usando int().

Exemplo 3 – Cálculo com entrada de dados


a = float(input("Digite o primeiro número: "))
b = float(input("Digite o segundo número: "))
print("Soma:", a + b)

Este exemplo mostra como receber dois valores numéricos, convertê-los e realizar uma operação matemática.
