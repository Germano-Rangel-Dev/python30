Dia 14 – Módulos e Bibliotecas em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

À medida que programas crescem, colocar todo o código em um único arquivo se torna confuso e difícil de manter. Para resolver isso, Python permite dividir o código em partes menores, organizadas e reutilizáveis, chamadas de módulos.

Além disso, Python possui um vasto ecossistema de bibliotecas, que são conjuntos de módulos prontos para uso, criados pela comunidade ou pela própria linguagem.

Conceito-chave:
Módulos e bibliotecas permitem reutilizar código e organizar projetos de forma profissional.

2️⃣ Explicação Técnica Detalhada

Um módulo é simplesmente um arquivo Python (.py) que contém funções, variáveis ou classes. Para utilizar um módulo, usamos a palavra-chave import.

Bibliotecas são conjuntos de módulos. Algumas já vêm instaladas com o Python, como math, random e datetime. Outras podem ser instaladas usando o pip.

Existem diferentes formas de importar módulos:

  • import modulo
  • from modulo import funcao
  • import modulo as apelido

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Usando um módulo padrão


import math

print(math.sqrt(25))

O módulo math fornece funções matemáticas prontas.

Exemplo 2 – Importando funções específicas


from random import randint

numero = randint(1, 10)
print(numero)

Aqui importamos apenas a função necessária do módulo random.

Exemplo 3 – Criando e usando um módulo próprio


# arquivo: util.py
def dobro(n):
    return n * 2


# arquivo principal
import util

print(util.dobro(5))

Este exemplo mostra como criar e reutilizar um módulo personalizado.
