Dia 10 – Listas em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Até agora, trabalhamos principalmente com variáveis que armazenam apenas um valor. Em aplicações reais, no entanto, é muito comum precisar armazenar e manipular vários valores relacionados, como notas de alunos, nomes de usuários ou registros de vendas.

Para resolver esse problema, Python oferece uma estrutura chamada lista. Listas permitem armazenar múltiplos valores em uma única variável, mantendo a ordem dos elementos e permitindo modificações ao longo do programa.

Conceito-chave:
Lista é uma estrutura de dados que armazena vários valores em uma única variável, de forma ordenada e mutável.

2️⃣ Explicação Técnica Detalhada

Uma lista é criada utilizando colchetes [], com os elementos separados por vírgula. Os elementos podem ser de qualquer tipo: números, textos, valores lógicos e até outras listas.

Cada elemento da lista possui um índice, que começa em 0. Isso significa que o primeiro elemento está na posição 0, o segundo na posição 1, e assim por diante.

Listas são mutáveis, ou seja, seus valores podem ser alterados após a criação, diferentemente de outros tipos de dados que veremos mais adiante.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Criando uma lista


numeros = [10, 20, 30, 40]
print(numeros)

Criamos uma lista com quatro números inteiros.

Exemplo 2 – Acessando elementos


nomes = ["Ana", "Bruno", "Carlos"]
print(nomes[0])
print(nomes[1])

O acesso aos elementos é feito pelo índice. O primeiro elemento sempre tem índice 0.

Exemplo 3 – Modificando e percorrendo a lista


notas = [7, 8, 6]
notas[1] = 9

for nota in notas:
    print(nota)

Neste exemplo, alteramos um valor da lista e depois usamos um laço for para percorrer todos os elementos.

6️⃣ Resolução dos Testes

Teste 2: O índice é 0.

Teste 3: Listas são mutáveis.

7️⃣ Resolução dos Desafios


# Desafio 1
nomes = ["Ana", "Bruno", "Carlos"]
for nome in nomes:
    print(nome)

# Desafio 2
numeros = [1, 2, 3, 4, 5, 6]
for n in numeros:
    if n % 2 == 0:
        print(n)

# Desafio 3
notas = [7, 8, 6, 9]
media = sum(notas) / len(notas)
print("Média:", media)
