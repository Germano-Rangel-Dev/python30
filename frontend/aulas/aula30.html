Dia 30 – Projeto Final + Certificação

Curso Python em 30 Dias

1️⃣ Encerramento do Curso

Você chegou ao último dia do curso. Ao longo de 30 dias, você percorreu desde os fundamentos da linguagem Python até a criação de aplicações completas, incluindo banco de dados, APIs, autenticação, frontend e boas práticas.

Neste dia final, você irá consolidar todo o conhecimento em um projeto final profissional, equivalente ao que é exigido em vagas de nível júnior a pleno.

Missão final:
Criar uma aplicação completa, funcional e organizada, integrando tudo o que foi aprendido.

2️⃣ Descrição do Projeto Final

O projeto final será uma Plataforma de Cadastro e Acesso, contendo:

  • Cadastro de usuários
  • Login com autenticação JWT
  • Banco de dados SQLite
  • API com FastAPI
  • Frontend consumindo a API
  • Área protegida (dashboard)

3️⃣ Estrutura Final do Projeto


projeto-final/
 ├── backend/
 │   ├── main.py
 │   ├── database.py
 │   ├── models.py
 │   ├── auth.py
 │   └── requirements.txt
 ├── frontend/
 │   ├── index.html
 │   ├── login.html
 │   ├── dashboard.html
 │   └── script.js
 └── README.md

4️⃣ Exemplo de Fluxo da Aplicação

  1. Usuário se cadastra
  2. Senha é salva com hash
  3. Login retorna token JWT
  4. Token é salvo no navegador
  5. Dashboard consome rota protegida

5️⃣ Testes de Conhecimento Final

Teste 1: Qual a função do JWT no sistema?

Teste 2: Por que separar backend e frontend?

Teste 3: Qual vantagem do uso de banco de dados?

6️⃣ Desafio Final (Obrigatório)

Implemente no projeto final:

  • Validação de formulário no frontend
  • Proteção total do dashboard
  • Logout funcional
  • Tratamento de erros no backend

7️⃣ Resolução Esperada

O projeto deve:

  • Rodar sem erros
  • Ter código organizado
  • Utilizar boas práticas
  • Demonstrar domínio do Python

8️⃣ Certificado de Conclusão

Após concluir o projeto final, o aluno está apto a receber o Certificado de Conclusão – Curso Python em 30 Dias.

O certificado deve conter:

  • Nome do aluno

  • QR Code de validação

Este certificado pode ser gerado automaticamente em PDF e integrado à plataforma.
