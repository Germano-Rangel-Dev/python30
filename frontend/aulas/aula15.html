Dia 15 – Manipulação de Arquivos em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Até este ponto do curso, todos os dados utilizados pelos programas existiam apenas enquanto o código estava em execução. Assim que o programa era encerrado, todas as informações eram perdidas. Em aplicações reais, isso não é suficiente.

Para que dados sejam mantidos entre execuções, é necessário armazená-los de forma permanente. Uma das formas mais simples de persistência de dados é através de arquivos. Python oferece ferramentas poderosas e simples para ler e escrever arquivos.

Conceito-chave:
Manipulação de arquivos permite salvar e recuperar dados de forma permanente no sistema de arquivos.

2️⃣ Explicação Técnica Detalhada

Em Python, a função principal para trabalhar com arquivos é open(). Ela permite abrir um arquivo em diferentes modos:

  • "r" – leitura (read)
  • "w" – escrita (write)
  • "a" – adicionar conteúdo (append)

Sempre que abrimos um arquivo, é fundamental fechá-lo após o uso. A forma mais segura de trabalhar com arquivos é utilizando a estrutura with, que garante o fechamento automático.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Escrevendo em um arquivo


with open("dados.txt", "w") as arquivo:
    arquivo.write("Olá, Python!")

Este código cria um arquivo (ou sobrescreve, se já existir) e grava um texto.

Exemplo 2 – Lendo um arquivo


with open("dados.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)

Aqui o conteúdo do arquivo é lido e exibido no terminal.

Exemplo 3 – Adicionando conteúdo


with open("dados.txt", "a") as arquivo:
    arquivo.write("\nNova linha adicionada")

O modo "a" adiciona conteúdo ao final do arquivo sem apagar o existente.
