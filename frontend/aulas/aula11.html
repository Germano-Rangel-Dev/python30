Dia 11 – Tuplas em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

À medida que avançamos no estudo das estruturas de dados, é importante entender que nem sempre precisamos de estruturas que possam ser modificadas. Em muitos casos, queremos apenas armazenar um conjunto fixo de valores que não deve ser alterado ao longo do programa.

Para esses casos, Python oferece a estrutura chamada tupla. Tuplas são muito parecidas com listas, mas possuem uma diferença fundamental: elas são imutáveis.

Conceito-chave:
Tupla é uma estrutura de dados ordenada e imutável usada para armazenar vários valores relacionados.

2️⃣ Explicação Técnica Detalhada

Tuplas são criadas utilizando parênteses () ou simplesmente separando valores por vírgula. Assim como listas, seus elementos possuem índices que começam em zero.

A principal característica das tuplas é a imutabilidade. Isso significa que, após criadas, não é possível adicionar, remover ou modificar elementos. Essa característica torna as tuplas mais seguras para representar dados fixos.

Tuplas são frequentemente utilizadas para:

  • representar coordenadas
  • retornar múltiplos valores de uma função
  • armazenar configurações fixas

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Criando uma tupla


cores = ("vermelho", "verde", "azul")
print(cores)

Criamos uma tupla com três elementos do tipo texto.

Exemplo 2 – Acessando elementos


coordenadas = (10, 20)
print(coordenadas[0])
print(coordenadas[1])

Os elementos da tupla são acessados pelo índice, assim como nas listas.

Exemplo 3 – Tuplas em operações práticas


pessoa = ("Germano", 30, "Programador")

for dado in pessoa:
    print(dado)

Apesar de imutáveis, tuplas podem ser percorridas usando laços de repetição.
