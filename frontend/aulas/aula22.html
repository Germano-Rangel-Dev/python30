Dia 22 – Banco de Dados com Python (SQLite)

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Até agora, utilizamos arquivos para armazenar dados. Embora funcionem bem para aplicações simples, arquivos não são ideais para sistemas maiores, com muitos usuários e consultas frequentes.

Para isso, utilizamos bancos de dados. Eles permitem armazenar, organizar, buscar e atualizar dados de forma rápida, segura e estruturada.

Conceito-chave:
Banco de dados é um sistema especializado para armazenar e gerenciar dados de forma eficiente.

2️⃣ Por que SQLite?

SQLite é um banco de dados leve, embutido e que não precisa de servidor. Ele é ideal para aprender bancos de dados e para aplicações pequenas e médias.

  • Não requer instalação complexa
  • Funciona com um único arquivo
  • Já vem com o Python

3️⃣ Primeiros Passos com SQLite

Exemplo 1 – Criando conexão


import sqlite3

conexao = sqlite3.connect("banco.db")
cursor = conexao.cursor()

Criamos uma conexão com o banco de dados.

Exemplo 2 – Criando tabela


cursor.execute("""
CREATE TABLE IF NOT EXISTS usuarios (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    nome TEXT,
    email TEXT,
    senha TEXT
)
""")
conexao.commit()

Criamos uma tabela estruturada para usuários.

Exemplo 3 – Inserindo dados


cursor.execute(
    "INSERT INTO usuarios (nome, email, senha) VALUES (?, ?, ?)",
    ("Germano", "email@email.com", "hash")
)
conexao.commit()

Inserção segura utilizando parâmetros.

Exemplo 4 – Consultando dados


cursor.execute("SELECT * FROM usuarios")
usuarios = cursor.fetchall()

for u in usuarios:
    print(u)
