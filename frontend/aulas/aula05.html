Dia 05 – Tipos de Dados em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Em programação, não basta apenas armazenar valores em variáveis. É necessário entender que tipo de informação cada valor representa. Os tipos de dados definem como o Python interpreta, armazena e manipula essas informações.

Compreender tipos de dados é essencial para evitar erros, escrever códigos mais claros e realizar operações corretamente. Um número, um texto e um valor lógico se comportam de maneiras completamente diferentes.

Conceito-chave:
Tipo de dado é a classificação do valor armazenado em uma variável, determinando quais operações podem ser realizadas sobre ele.

2️⃣ Explicação Técnica Detalhada

Python possui diversos tipos de dados nativos. Os principais para iniciantes são:

  • int – números inteiros
  • float – números decimais
  • str – textos
  • bool – valores lógicos (True ou False)

Python identifica automaticamente o tipo de dado no momento da atribuição. Você pode verificar o tipo de qualquer valor usando a função type().

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Tipos básicos


idade = 30        # int
altura = 1.75    # float
nome = "Germano" # str
ativo = True     # bool

print(type(idade))
print(type(altura))
print(type(nome))
print(type(ativo))

Este exemplo mostra como o Python identifica automaticamente o tipo de cada variável.

Exemplo 2 – Operações com tipos numéricos


a = 10
b = 3

print(a + b)
print(a / b)
print(type(a / b))

Mesmo dividindo dois inteiros, o resultado é um número decimal (float).

Exemplo 3 – Combinação de texto e números


idade = 30
print("Idade:", idade)

Python permite combinar texto e números no print, desde que cada elemento esteja separado por vírgula ou formatado corretamente.
