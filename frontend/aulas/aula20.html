A partir deste ponto, começamos a construir aplicações completas. Neste projeto, você irá desenvolver um sistema de cadastro, utilizando tudo o que foi aprendido até agora: variáveis, estruturas de dados, funções, arquivos, tratamento de erros e Programação Orientada a Objetos.

Objetivo do projeto:
Criar um sistema capaz de cadastrar, listar e salvar usuários em arquivo.

2️⃣ Estrutura do Sistema

O sistema será dividido logicamente em:

  • Classe para representar o usuário
  • Funções para cadastrar e listar usuários
  • Persistência em arquivo (JSON)
  • Menu interativo

3️⃣ Implementação Passo a Passo

Passo 1 – Classe Usuario


class Usuario:
    def __init__(self, nome, email):
        self.nome = nome
        self.email = email

    def to_dict(self):
        return {
            "nome": self.nome,
            "email": self.email
        }

Passo 2 – Salvando usuários em arquivo


import json

def salvar_usuario(usuario):
    try:
        with open("usuarios.json", "r") as f:
            usuarios = json.load(f)
    except:
        usuarios = []

    usuarios.append(usuario.to_dict())

    with open("usuarios.json", "w") as f:
        json.dump(usuarios, f, indent=4)

Passo 3 – Listando usuários


def listar_usuarios():
    try:
        with open("usuarios.json", "r") as f:
            usuarios = json.load(f)
            for u in usuarios:
                print(u["nome"], "-", u["email"])
    except:
        print("Nenhum usuário cadastrado.")

Passo 4 – Menu principal


while True:
    print("1 - Cadastrar")
    print("2 - Listar")
    print("0 - Sair")

    opcao = input("Escolha: ")

    if opcao == "1":
        nome = input("Nome: ")
        email = input("Email: ")
        usuario = Usuario(nome, email)
        salvar_usuario(usuario)

    elif opcao == "2":
        listar_usuarios()

    elif opcao == "0":
        break
