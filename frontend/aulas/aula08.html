Dia 08 – Condicionais em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Até agora, todos os programas executavam sempre as mesmas instruções, do início ao fim. No mundo real, porém, programas precisam tomar decisões com base em condições, valores e escolhas do usuário.

As estruturas condicionais permitem que um programa execute diferentes blocos de código dependendo de uma condição ser verdadeira ou falsa. Em Python, isso é feito principalmente com if, elif e else.

Conceito-chave:
Condicionais permitem que um programa execute caminhos diferentes conforme uma condição avaliada como verdadeira ou falsa.

2️⃣ Explicação Técnica Detalhada

A estrutura básica de uma condicional em Python é: 


if condição:
    # código executado se a condição for verdadeira

Caso a condição seja falsa, o código dentro do if não será executado. Para tratar outras possibilidades, utilizamos elif (else if) e else.

A indentação é obrigatória em Python e define quais linhas pertencem a cada bloco condicional. Erros de indentação são uma das principais causas de problemas para iniciantes.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Condicional simples


idade = 18

if idade >= 18:
    print("Maior de idade")

O código só será executado se a condição for verdadeira.

Exemplo 2 – if / else


idade = 16

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

O else cobre o caso em que a condição do if é falsa.

Exemplo 3 – if / elif / else


nota = 7

if nota >= 9:
    print("Excelente")
elif nota >= 6:
    print("Aprovado")
else:
    print("Reprovado")

O elif permite testar múltiplas condições em sequência.
