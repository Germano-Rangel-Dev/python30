Dia 24 – APIs com Python (FastAPI)

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Até agora, seus programas rodavam localmente no terminal. Porém, aplicações modernas funcionam conectadas, trocando dados entre sistemas, sites e aplicativos. Essa comunicação acontece através de APIs.

Uma API permite que outros sistemas se comuniquem com seu código Python por meio da internet, utilizando requisições HTTP.

Conceito-chave:
API é uma interface que permite que sistemas se comuniquem via requisições HTTP.

2️⃣ O que é FastAPI?

FastAPI é um framework moderno e rápido para criar APIs em Python. Ele é amplamente utilizado no mercado por sua performance, simplicidade e documentação automática.

  • Alta performance
  • Fácil de aprender
  • Documentação automática

3️⃣ Criando a Primeira API

Passo 1 – Instalação


pip install fastapi uvicorn

Passo 2 – API básica


from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def home():
    return {"mensagem": "API funcionando!"}

Passo 3 – Rodando o servidor


uvicorn main:app --reload

A API ficará disponível em http://127.0.0.1:8000.

4️⃣ Rotas e Métodos HTTP

As rotas definem os caminhos da API. Os métodos HTTP mais comuns são:

  • GET – Buscar dados
  • POST – Criar dados
  • PUT – Atualizar dados
  • DELETE – Remover dados

@app.get("/usuarios")
def listar_usuarios():
    return ["Ana", "Bruno"]
