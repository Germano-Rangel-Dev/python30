Dia 03 – Primeiro Programa em Python

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Escrever o primeiro programa é um marco importante para qualquer pessoa que está aprendendo a programar. Neste momento, conceitos abstratos passam a fazer sentido na prática: o código é escrito, executado e produz um resultado visível.

Neste dia, você aprenderá como criar um arquivo Python, como o interpretador executa esse arquivo, como funciona o fluxo de execução do código e quais são os erros mais comuns nesse primeiro contato.

Conceito-chave:
Um programa Python é um conjunto de instruções escritas em um arquivo .py, executadas de cima para baixo pelo interpretador.

2️⃣ Explicação Técnica Detalhada

Arquivos Python utilizam a extensão .py. Quando você executa um arquivo, o interpretador lê cada linha na ordem em que foi escrita e executa as instruções sequencialmente.

Isso significa que a ordem das linhas importa. Uma instrução escrita antes de outra será executada primeiro. Essa característica é conhecida como fluxo sequencial.

Outro ponto importante é a função print(), responsável por exibir informações no terminal. Ela é essencial para testar e entender o comportamento do código.

3️⃣ Exemplos Práticos (do básico ao avançado)

Exemplo 1 – Primeiro programa


print("Meu primeiro programa em Python")

Este código simplesmente exibe uma mensagem no terminal. É a forma mais simples de verificar se tudo está funcionando corretamente.

Exemplo 2 – Ordem de execução


print("Linha 1")
print("Linha 2")
print("Linha 3")

As mensagens serão exibidas exatamente na ordem em que aparecem no código, mostrando que Python executa o programa de cima para baixo.

Exemplo 3 – Mensagens explicativas


print("Iniciando programa")
print("Processando dados")
print("Programa finalizado")

Esse padrão é muito usado para acompanhar o andamento de um programa, facilitando a depuração e o entendimento do fluxo.
