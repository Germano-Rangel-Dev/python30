Dia 23 – CRUD Completo com Banco de Dados

Curso Python em 30 Dias

1️⃣ Introdução Conceitual

Agora que você já sabe criar e acessar um banco de dados, o próximo passo é implementar um CRUD completo. CRUD é um acrônimo para Create, Read, Update e Delete, que representam as quatro operações básicas de qualquer sistema com banco de dados.

Todo sistema profissional — como cadastros, e-commerces, ERPs e APIs — utiliza CRUD para gerenciar informações.

Conceito-chave:
CRUD representa as operações fundamentais para gerenciar dados em um banco.

2️⃣ Estrutura do CRUD

  • Create – Criar registros
  • Read – Ler registros
  • Update – Atualizar registros
  • Delete – Remover registros

3️⃣ Implementação Passo a Passo

Create – Inserir dados


def criar_usuario(nome, email, senha):
    cursor.execute(
        "INSERT INTO usuarios (nome, email, senha) VALUES (?, ?, ?)",
        (nome, email, senha)
    )
    conexao.commit()

Read – Listar usuários


def listar_usuarios():
    cursor.execute("SELECT * FROM usuarios")
    return cursor.fetchall()

Update – Atualizar usuário


def atualizar_usuario(id, novo_nome):
    cursor.execute(
        "UPDATE usuarios SET nome = ? WHERE id = ?",
        (novo_nome, id)
    )
    conexao.commit()

Delete – Remover usuário


def deletar_usuario(id):
    cursor.execute(
        "DELETE FROM usuarios WHERE id = ?",
        (id,)
    )
    conexao.commit()
